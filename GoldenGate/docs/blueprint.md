## Core Features:

- Dashboard: Elegant dashboard with role-based access (Admin, Cashier, Seller).
- Inventory Management: Manage inventory for Gold, Silver, and Jewelry (rings, watches, tiaras, etc.) with image support.
- Financing Management: Manage financing (layaway) with customer and debt records.
- Reporting: Export reports to Excel/CSV format.

## Style Guidelines:

- Primary color: Deep Gold (#D4AF37) to reflect the brand's luxury and sophistication.
- Background color: Dark Charcoal (#333333) for a sleek and modern backdrop with 20% saturation of the hue from the primary color.
- Accent color: Light Gold (#B8860B) to highlight interactive elements and call-to-actions, with a saturation and brightness distinctly different than the primary.
- Headline font: 'Playfair', a serif font, for headlines. This matches the user's request.
- Body font: 'PT Sans', a sans-serif font, to ensure readability and complement Playfair. It will be paired with Playfair as requested by the user. Note: currently only Google Fonts are supported.
- Use refined, minimalist icons for navigation and actions.
- Responsive design ensuring usability across devices.
- Subtle transitions and animations for a polished user experience.
